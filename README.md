# Target Auto-Checkout Extension

Browser extension for automated Target.com checkout with Discord integration.

## Features

### Core Functionality
- Automated checkout for Target.com products
- Fast mode with optimized delays (5ms polling)
- Profile management for quick checkout
- CVV and payment verification
- Price checking with configurable limits

### Discord Integration (NEW!)
- Monitor Discord channels for product links
- Auto-open Target product links from Discord
- Smart cooldowns to prevent spam
- Multi-channel support
- Browser notifications

### Performance Optimizations (LATEST: v3.7!)
- **12.5x faster** product page activation (2500ms → 200ms)
- **8x faster** quantity changes (1200ms → 150ms)
- **6.7x faster** form filling (2000ms → 300ms)
- **4.2x faster** login process (3800ms → 900ms)
- **4.5x faster** overall checkout (~9s → ~2s)
- Ultra-fast polling for immediate response (5ms intervals)
- Optimized for winning high-demand items

## Quick Start

1. Load extension in Chrome/Edge
2. Configure profile in options
3. Set quantity and enable for Target
4. (Optional) Set up Discord monitoring
5. Navigate to Target product page
6. Extension handles the rest!
